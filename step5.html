Хранилище объектов

В C++ Builder присутствует удобный инструмент — "Хранилище объектов". Здесь представлено множество шаблонов форм, диалогов, типов файлов, а иногда даже целых проектов. Доступ к Хранилищу осуществляется через команду File->New, и его интерфейс выглядит следующим образом:
Спроектируйте формы по образцу:

На вкладках доступны полезные шаблоны, включая MDI Application из раздела Projects.
Этот шаблон предназначен для создания приложения с интерфейсом многодокументного окружения (MDI), например, подобного MS Word. Запустите мастер, укажите местоположение проекта и завершите создание.
Полученный шаблон включает меню, панель статуса и инструментов. Давайте создадим мультимедийный проигрыватель, добавив TToolBar из вкладки Win32 и поверх него TMediaPlayer из вкладки System. Отредактируйте MDIChild, удалив ненужный компонент Memo1.
